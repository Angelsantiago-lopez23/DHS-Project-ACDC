Property Tax Web Scraping Script README

Overview

This script automates the task of extracting detailed information about property taxes from a specific website. It utilizes the names provided in an Excel file to search for corresponding account, owner, and address details, which are then compiled into another Excel file.

Prerequisites

  • Python 3.x installed.
  • Google Chrome Browser.
  • Internet connection.

Installation

  1. Python Libraries: Install the required Python libraries by running: pip install selenium pandas webdriver-manager
  2. Chrome WebDriver: The script uses ChromeDriver, which is automatically managed by webdriver-manager.

Input File Preparation

- The input must be an Excel file named input.xlsx.

- Ensure it has a column with the header name.

Execution Steps

  1. Place the input.xlsx file in the same directory as the script.
  2. Run the script: python <script_name>.py
    Replace <script_name> with the actual file name of the Python script.

Post-execution

- Upon completion, the script generates a file named results.xlsx in the same directory, containing the extracted data.

Troubleshooting

  • Script not starting: Check if all prerequisites are met.
  • Browser issues: Ensure the latest version of Chrome is installed.
  • Excel file errors: Verify the format of input.xlsx as mentioned in the guidelines.

Notes for Testers

  • Conduct tests with various names to ensure the script's robustness.
  • Monitor the performance and log any issues encountered.
  • Verify the accuracy and completeness of data in the output file.

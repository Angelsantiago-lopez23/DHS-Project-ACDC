Web Scraping Script README

Overview

This script automates the process of extracting property information from a specific website. It reads names from an Excel file and searches for corresponding property details, which are then saved into another Excel file.

Prerequisites

  • Python 3.x installed.
  • Google Chrome Browser.
  • Internet connection.

Installation

  1. Python Libraries: Install necessary Python libraries by running: pip install selenium pandas webdriver-manager
  2. Chrome WebDriver: The script uses ChromeDriver, which will be automatically managed by webdriver-manager.

Input File Preparation

- The input should be an Excel file named input.xlsx.

- Ensure it has a single column with the header name.

Execution Steps

  1. Place the input.xlsx file in the same directory as the script.
  2. Run the script: python <script_name>.py
    Replace <script_name> with the name of the Python file.

Post-execution

- After the script completes, it will generate an results.xlsx file in the same directory.

- This file contains the extracted data in structured format.

Troubleshooting

  • Script not starting: Ensure all prerequisites are met.
  • Browser issues: Verify that the latest version of Chrome is installed.
  • Excel file errors: Confirm the format of input.xlsx as per the guidelines.

Notes for Testers

  • Test with various input names to ensure robustness.
  • Monitor the script's performance and log any anomalies.
  • Check the output file for accuracy and completeness of data.
